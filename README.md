This is a collection of Data Structures and Algorithms (DSA) solutions from LeetCode problems, implemented in **Java**.

## About This Project

This repository contains my solutions to various **LeetCode** problems.
All solutions are organized in text files for easy reference and learning.

Each solution includes:

- Problem number and title
- Direct link to the LeetCode problem
- Java implementation with comments

## How to Use

1. Browse the .txt files to view solutions.
2. Each solution includes a direct LeetCode problem link.
3. Solutions are in Java and include explanatory comments.

```java
// [Problem Number]. [Problem Title]
// [LeetCode URL]

    // Implementation with comments
}
```
